About TaskTrek

TaskTrek is a modern and intuitive web application for task management, developed using Angular. It allows you to effectively organize your tasks, track progress, and stay productive.

This project demonstrates the use of Angular Standalone Components, Reactive Forms, dynamic routing, and interaction with a remote API for data storage.
